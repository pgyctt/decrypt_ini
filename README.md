# 对称字节变换工具 (decrypt-ini)

一个支持文件拖拽的 INI 文件加密/解密工具。

## 功能特性

- 🔒 **编码（加密）**: 使用对称字节变换算法加密文件
- 🔓 **解码（解密）**: 将加密文件还原为原始内容
- 🖱️ **文件拖拽**: 支持直接拖拽文件到窗口进行处理
- 📊 **实时预览**: 显示处理结果和文件信息
- 💾 **保存功能**: 将处理后的内容保存到文件

## 安装依赖

### 基本运行（无需额外依赖）
```bash
uv run python decrypt_ini.py
```

### 启用拖拽功能（推荐）
```bash
uv add tkinterdnd2
```

## 使用方法

### 方式一：拖拽文件（需要安装 tkinterdnd2）
1. 运行程序 `uv run python decrypt_ini.py`
2. 将文件直接拖拽到程序窗口中
3. 点击“编码”或“解码”按钮进行处理
4. 点击“保存到文件”保存结果

### 方式二：手动选择文件
1. 运行程序 `uv run python decrypt_ini.py`
2. 点击“浏览”按钮选择要处理的文件
3. 点击“编码”或“解码”按钮进行处理
4. 点击“保存到文件”保存结果

## 算法说明

- **编码算法**: `(byte ^ 0x36) >> 3 | ((byte ^ 0x36) & 0x07) << 5`
- **解码算法**: `((byte << 3) | (byte >> 5)) ^ 0x36`

这是一个对称的字节变换算法，编码和解码互为逆运算。

## 系统要求

- Python 3.14+
- Tkinter（通常随 Python 一起安装）
- tkinterdnd2（用于拖拽功能）

## 打包为可执行文件

项目提供了 `publish.bat` 脚本，可以将程序打包为独立的 Windows 可执行文件（.exe）。

### 打包步骤

1. **确保依赖已安装**：
   ```bash
   uv sync
   ```

2. **运行打包脚本**：
   ```bash
   .\publish.bat
   ```

3. **获取可执行文件**：
   - 打包完成后，可在 `dist` 目录下找到 `decrypt_ini.exe`
   - 该文件包含了所有依赖，可以独立运行，无需安装 Python

### 打包说明

- 使用 PyInstaller 进行打包
- `-F` 参数：生成单个可执行文件
- `-w` 参数：无控制台窗口（GUI 模式）
- `-i` 参数：使用自定义图标（assets\icon.ico）
- `--splash` 参数：显示启动画面（assets\loading.png）

### 注意事项

- 首次打包前需要安装 PyInstaller：`uv add pyinstaller`
- 打包过程可能需要几分钟时间
- 生成的 exe 文件较大（包含 Python 运行时和所有依赖）
- 杀毒软件可能会误报，这是 PyInstaller 打包的正常现象

## 注意事项

- 如果没有安装 `tkinterdnd2`，程序仍可正常运行，但不支持拖拽功能
- 编码后的文件是二进制格式，建议保存为 `.ini` 或其他扩展名
- 解码后的文件会根据内容尝试以 UTF-8 编码显示
